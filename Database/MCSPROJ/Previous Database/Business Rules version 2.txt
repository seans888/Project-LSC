Loyola Student Center Enrollment System
Business Rules:


1) Each STUDENT creates an ACCOUNT
	-ACCOUNT is being created by one or more STUDENT
	
2) Each STUDENT can enrolls/reserves/ one or more REVIEW CLASS
	-Each REVIEW CLASS enrolled/resered by one or more STUDENT
	
3) A REVIEW CLASS includes one or more PAYMENT
	-Each PAYMENT includes specific REVIEW CLASS

4) A REVIEW CLASS consists one or more SCHEDULE
	-Each SCHEDULE consists specific REVIEW CLASS

5) A STUDENT pays one or more PAYMENT
	-Each PAYMENT is being paid by the STUDENT
	
6) 	A REVIEW CLASS composes many CLASS LIST
	-Each CLASS LIST is being composed of REVIEW CLASS
	
7) 	A STUDENT contains in a CLASS LIST
	-Each CLASS LIST is being contained by the STUDENT
	
8) Each STUDENT receives one SCHEDULE
	-SCHEDULE is being received by a STUDENT
